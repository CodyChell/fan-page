PC Building Guide Fan-Page

I love building PC's! They are so much fun to build and make your own.

First I can talk about a few things about PC building.
A few things to start out what you need to buy.

Parts required for building a PC.

  1. Processor
  2. CPU Cooler
  3. Motherboard
  4. RAM
  5. Graphics Card
  6. Hard drive(s)
  7. Power Supply
  8. Case
  9. Operating System

Optional PC parts to buy.

Now that you have the require items that you need. There are some optional items that you can get while you're building your PC.
  • SSD for hard drive replacement. I highly reccommend getting this part along side a taditional rotating spinning drive.
  • Keyboard and Mouse replacement. Something like a mechanical keyboard.
  • Network card if necessary.
  • Monitor
  • Optical drive (You probably don't need this)
Since these are all optional items you don't have to get them. Some though, are highly reccommended.
